FoSpy.blocks.metadata

This site only contains documentation for the Block subclasses defined in this module, not the entire module API. For a complete reference of all functions, classes, and variables, see the full API documentation.

Methods attached to any object

When assigning an object as an attribute, SingleBlocks attach special methods that are used to edit their own comment metadata.

add_comments(self_attr, *comments)

Method attached to any SingleBlock attribute as add_comments(self_attr, *comments). Mutates the parent SingleBlock's comments metadata under the assigned attribute name.

Example 
my_synthesis.materials.add_comments("A comment on the materials header")
my_synthesis._meta.comments
# returns: 
#       {"materials":
#           [
#               "an older comment", 
#               "A comment on the materials header"
#           ]
#       }

Parameters:

Name Type Description Default
*comments str

Comments to attach to the assigned attribute name in the parent block's metadata. Comments are printed above their attached attribute when saving in the FOS format.

def _add_comments_to_parent(attr_name):
    """
    Method attached to any [`SingleBlock`][FoSpy.blocks.blocks.SingleBlock]
    attribute as `add_comments(self_attr, *comments)`. Mutates the parent
    `SingleBlock`'s comments metadata under the assigned attribute name.

    Example:
        ```
        my_synthesis.materials.add_comments("A comment on the materials header")
        my_synthesis._meta.comments
        # returns: 
        #       {"materials":
        #           [
        #               "an older comment", 
        #               "A comment on the materials header"
        #           ]
        #       }
        ```

    Args:
        *comments (str):
            Comments to attach to the assigned attribute name in
            the parent block's metadata. Comments are printed above
            their attached attribute when saving in the FOS format.
    """
    def func(self_attr, *comments):
        parent_comments = self_attr._parent_block._meta.comments
        parent_comments.setdefault(attr_name,[])
        for comment in comments:
            parent_comments[attr_name].append(str(comment))
    return func

clear_comments(self_attr)

Method attached to any SingleBlock attribute as clear_comments(self_attr). Clears the parent SingleBlock's comments metadata under the assigned attribute name.

Example 
my_synthesis._meta.comments["materials"]
# returns ["This comment was read from a FOS file"]

my_synthesis.materials.clear_comments()
my_synthesis._meta.comments["materials"]
# now returns []

def _clear_comments_from_parent(attr_name):
    """
    Method attached to any [`SingleBlock`][FoSpy.blocks.blocks.SingleBlock]
    attribute as `clear_comments(self_attr)`. Clears the parent
    `SingleBlock`'s comments metadata under the assigned attribute name.

    Example:
        ```
        my_synthesis._meta.comments["materials"]
        # returns ["This comment was read from a FOS file"]

        my_synthesis.materials.clear_comments()
        my_synthesis._meta.comments["materials"]
        # now returns []
        ```
    """
    def func(self_attr):
        parent_comments = self_attr._parent_block._meta.comments
        parent_comments[attr_name] = []
    return func

class Reaction(SingleBlock):
    """
    Represents information pertaining to the entire synthesis but not pertaining
    to FOS file metadata, such as nominal_mass or target_composition.
    """

    def get_nom_MW(self):
        """Returns the molecular weight of nominal_formula"""
        return self.nominal_formula.formula_weight

    @_calc_routine()
    def add_nom_MW(self):
        """
        Attaches a comment to the nominal_formula with the molecular weight.
        """
        mw = self.get_nom_MW()
        _debug.msg(f"Calculated Nominal MW: {mw:.2f} for Reaction block.")
        self.add_calc_comment("nominal_formula",f"Nominal MW: {mw:.2f} g/mol","Nominal MW")
